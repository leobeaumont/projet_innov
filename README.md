# Assistant Nutrition & Sport (LLM)

Script Python utilisant **LangChain** et **Gemini 2.5 Flash Lite** pour analyser la consommation calorique et l'activité physique.

## Installation
1. Cloner le repo : `https://github.com/leobeaumont/projet_innov`
2. Créer un venv : `python -m venv venv`
3. Installer les dépendances : `pip install -r requirements.txt`
4. Créer un fichier `.env` avec votre `GOOGLE_API_KEY`.

## Tech Stack
- LangChain
- Pydantic (pour le JSON structuré)
- Google Gemini API
